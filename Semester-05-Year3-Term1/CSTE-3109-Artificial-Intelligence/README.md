# 🤖 CSTE-3109: Artificial Intelligence

Welcome to the Artificial Intelligence repository for the 5th semester.

## 🎯 Course Objectives
* Introduction to theoretical and practical AI concepts, including intelligent agents (PEAS).
* Master informed and uninformed search strategies (BFS, DFS, A*, Hill-climbing).
* Understand adversarial search mechanisms (Minimax, Alpha-Beta Pruning).
* Delve into Knowledge Representation through Propositional and First-Order Logic.
* Learn uncertain reasoning via Probability, Bayes' Theorem, and Fuzzy Logic.
* Provide an introduction to modern Machine Learning algorithms (Decision Trees, ANNs, CNNs).

## 📚 Recommended Reference Materials
1. **Artificial Intelligence: A Modern Approach** by Stuart Russell and Peter Norvig.
2. **Prolog Programming for Artificial Intelligence** by Ivan Bratko.

## 💻 Lab Setup & Projects
The laboratory component (`CSTE-3110`) bridges theoretical AI logic with applied coding.
* **Primary Language:** Python or Prolog.
* **Key Tasks:** Implementing basic graph search algorithms, constraint satisfaction problem (CSP) solvers, genetic algorithms, and naive Bayesian classifiers.
