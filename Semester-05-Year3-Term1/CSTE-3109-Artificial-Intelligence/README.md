# 🤖 CSTE 3109: Artificial Intelligence
**Credit:** 3.0 | **Prerequisite:** CSTE 2103

> *"Theoretical and practical AI approaches (agents, search, logic, ML)."*

</div>

---

## 🎯 Course Objectives
*   Introduction to theoretical and practical AI concepts, including intelligent agents (PEAS).
*   Master informed and uninformed search strategies (BFS, DFS, A*, Hill-climbing).
*   Understand adversarial search mechanisms (Minimax, Alpha-Beta Pruning).
*   Delve into Knowledge Representation through Propositional and First-Order Logic.
*   Learn uncertain reasoning via Probability, Bayes' Theorem, and Fuzzy Logic.
*   Provide an introduction to modern Machine Learning algorithms.

## 📅 Weekly Lesson Plan

| Week | Course Contents |
| :---: | :--- |
| **1** | Intro to AI: Definition, History, Intelligent Agents (PEAS). |
| **2** | Problem Solving by Searching: Uninformed Search (BFS, DFS). |
| **3** | Informed (Heuristic) Search: A*, Greedy, Local Search (Hill-climbing). |
| **4** | Genetic Algorithms (GA): Cycle, Selection, Crossover, Mutation. |
| **5** | Adversarial Search (Game Playing): Minimax, Alpha-Beta Pruning. |
| **6** | Constraint Satisfaction Problems (CSP): Backtracking, Propagation. |
| **7** | Knowledge Representation & Logic I: Propositional Logic, Inference. |
| **8** | Knowledge Representation & Logic II: First-Order Logic (FOL), Unification. |
| **9** | Uncertain Knowledge & Reasoning: Probability, Bayes’ Theorem, Bayesian Networks. |
| **10** | Fuzzy Logic: Fuzzy Sets, Relations, Fuzzy Logic Controller. |
| **11** | Probabilistic Reasoning over Time: HMM, Kalman Filters. |
| **12** | Machine Learning Introduction: Supervised Learning, Decision Trees, ANN, CNN. |
| **13** | Review & Trends: Planning, NLP, Robotics, Ethics. |

## 📚 Recommended Books

2. 📖 *Prolog Programming for Artificial Intelligence* by Ivan Bratko
