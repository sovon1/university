## 📘 Course Details

### 1. CSTE 3101: Digital Signal Processing
*   **Credit:** 3.0
*   **Prerequisite:** CSTE 2203 (Signals and Systems)
*   **Rationale:** Deals with digital signals, systems, applications, operations, and digital filter design.
*   **Objectives:**
    *   Introduce signals, systems, time and frequency domain concepts.
    *   Discuss Z-transform and discrete-time Fourier transform.
    *   Discuss sampling techniques and reconstruction.
    *   Explain functionalities of digital filters (FIR/IIR).

**Weekly Lesson Plan:**
| Week | Course Contents |
| :--- | :--- |
| **1** | Discrete Time Signals & Systems: Intro, sequences, classification, LTI systems analysis. |
| **2** | Convolution & Correlation: Linear/Circular convolution, Difference equations. |
| **3** | Sampling: Sampling theorem, reconstruction, Aliasing, Quantization noise. |
| **4** | Frequency Domain Analysis: DTFT, Energy density spectrum. |
| **5** | Z-Transform: ROC, Properties, Rational Z-transforms (poles/zeros). |
| **6** | Inverse Z-Transform & Stability: Partial fraction, Jury’s stability criterion. |
| **7** | Discrete Fourier Transform (DFT): DFS, Properties, Linear filtering. |
| **8** | Fast Fourier Transform (FFT): Radix-2 algorithms, Butterfly diagrams. |
| **9** | Structures for Discrete-Time Systems: Block diagrams, IIR structures (Direct, Cascade, Parallel). |
| **10** | Structures for FIR Systems: Direct, Cascade, Lattice structure, Linear-phase. |
| **11** | Filter Design (IIR): Impulse invariance, Bilinear transformation, Butterworth, Chebyshev. |
| **12** | Filter Design (IIR) Continued: Frequency transformations, Low-pass/High-pass design. |
| **13** | Filter Design (FIR): Windowing method, Frequency sampling, Equiripple designs. |

**Recommended Books:**
1. *Digital Signal Processing: Principles, Algorithms, and Applications* by J.G. Proakis & D.G. Manolakis
2. *Understanding Digital Signal Processing* by R. G. Lyon

### 2. CSTE 3103: Operating Systems and System Programming
*   **Credit:** 3.0
*   **Prerequisite:** CSTE 2103
*   **Rationale:** Develops ability to realize OS functionalities (process, memory, file systems, security).

**Weekly Lesson Plan:**
| Week | Course Contents |
| :--- | :--- |
| **1** | Intro to OS: Types (Mainframe, Desktop, Real-time), OS Structures, System Calls. |
| **2** | Process Management: Process concept, Scheduling, IPC, Threads (Pthreads). |
| **3** | CPU Scheduling: FCFS, SJF, Priority, Round Robin, Multilevel Queue. |
| **4** | Process Synchronization: Critical-Section, Semaphores, Dining-Philosophers, Monitors. |
| **5** | Deadlocks: Characterization, Prevention, Avoidance (Banker’s Algo), Detection. |
| **6** | Memory Management: Swapping, Paging, Segmentation, Fragmentation. |
| **7** | Virtual Memory: Demand Paging, Page Replacement (FIFO, LRU), Thrashing. |
| **8** | File System Interface: Access Methods, Directory Structure, Protection. |
| **9** | File System Implementation: Allocation Methods, Free-Space Management, Recovery. |
| **10** | Mass-Storage Structure: Disk Scheduling, RAID Structure (Levels 0, 1, 5, 10). |
| **11** | I/O Systems: Polling, Interrupts, DMA, Kernel I/O Subsystem. |
| **12** | Protection and Security: Access Matrix, Threats (Viruses, Worms, DoS). |
| **13** | Cryptography & System Programming: Authentication, Assemblers, Linkers, Macro Processors. |

**Recommended Books:**
1. *Operating System Concepts* by Silberschatz, Galvin, Gagne
2. *Modern Operating Systems* by Andrew S. Tanenbaum

### 3. EEE 3101: Electromagnetic Waves and Radiating Systems
*   **Credit:** 3.0
*   **Prerequisite:** PHYS 1101
*   **Rationale:** Background in electromagnetic theory, Maxwell equations, wave propagation, and radiation.

**Weekly Lesson Plan:**
| Week | Course Contents |
| :--- | :--- |
| **1** | Maxwell’s Equations: Vector Analysis, Faraday’s Law, Boundary Conditions. |
| **2** | EM Wave Propagation: Plane waves, Skin Depth, Poynting Vector, Reflection. |
| **3** | Guided Waves (Parallel Planes): TE, TM, TEM waves, Cut-off frequency. |
| **4** | Rectangular Waveguides: TE/TM modes, Dominant Mode, Power Transmission. |
| **5** | Transmission Line Theory I: Parameters (R,L,G,C), Wave Propagation, Characteristic Impedance. |
| **6** | Transmission Line Theory II: Input Impedance, SWR, Reflection Coefficient. |
| **7** | Impedance Matching: Quarter-wave transformer, Stub matching, Smith Chart. |
| **8** | Radiation and Potentials: Retarded Potentials, Hertzian Dipole, Radiation Resistance. |
| **9** | Antenna Fundamentals: Parameters (Gain, Directivity), Friis Transmission Equation. |
| **10** | Antenna Arrays: Linear Arrays, Broadside/End-fire, Pattern Multiplication. |
| **11** | Practical Antennas I: Dipole, Monopole, Loop, Yagi-Uda Antenna. |
| **12** | Practical Antennas II: Slot, Horn, Parabolic Reflector, LPDA. |
| **13** | Wave Propagation: Ground, Sky (Ionospheric), Space Wave Propagation. |

**Recommended Books:**
1. *Electromagnetic Waves and Radiating Systems* by E.C. Jordan & K.G. Balmain
2. *Engineering Electromagnetics* by W.H. Hayt & J.A. Buck

### 4. CSTE 3105: Microprocessor, Microcontroller and Interfacing
*   **Credit:** 3.0
*   **Prerequisite:** CSTE 2207
*   **Rationale:** Intro to microprocessor architecture (8086), assembly, and peripheral interfacing.

**Weekly Lesson Plan:**
| Week | Course Contents |
| :--- | :--- |
| **1** | Microprocessor Fundamentals: 8086 Architecture (BIU, EU), Registers. |
| **2** | Memory Organization & Addressing: Segmentation, Physical Memory, Pin Diagram. |
| **3** | Addressing Modes & Instruction Formats: Data, Program, Stack addressing. |
| **4** | Assembly Language Programming (Data Transfer): MOV, PUSH, POP, String Instructions. |
| **5** | Arithmetic & Logic Instructions: ADD, SUB, MUL, DIV, Logic, Shift/Rotate. |
| **6** | Program Control Instructions: Jumps, Loops, Procedures, Stack operations. |
| **7** | I/O Interfacing & Interrupts: I/O mapped vs Memory mapped, 8259A PIC. |
| **8** | Peripheral Interfacing I: 8255A PPI, Interfacing LEDs, Displays, Keyboards. |
| **9** | Peripheral Interfacing II: 8254 Timer, DMA Controller (8237). |
| **10** | Advanced Interfacing: ADC/DAC, Stepper Motor, RS-232C, USART. |
| **11** | Microcontrollers (8051): Architecture, Registers, I/O Ports, Memory. |
| **12** | 8051 Programming & Features: Instruction Set, Timers, Serial Comm, Interrupts. |
| **13** | Modern Microcontrollers: AVR (ATmega32), ARM, RISC features, Embedded Systems. |

**Recommended Books:**
1. *Microprocessors and Interfacing: Programming and Hardware* by Douglas V. Hall
2. *The Intel Microprocessors* by Barry B. Brey

### 5. CSTE 3107: Compiler Construction
*   **Credit:** 3.0
*   **Prerequisite:** CSTE 2107
*   **Rationale:** Understanding compiler phases, automata theory, and constructing a compiler.

**Weekly Lesson Plan:**
| Week | Course Contents |
| :--- | :--- |
| **1** | Intro to Compilers: Phases (Lexical, Syntax, etc.), Tools. |
| **2** | Lexical Analysis: Tokens, Regular Expressions, Finite Automata, Lex/Flex. |
| **3** | Syntax Analysis (Parsing) I: CFG, Parse Trees, Top-Down Parsing (LL(1)). |
| **4** | Syntax Analysis (Parsing) II: Bottom-Up Parsing, Shift-Reduce, LR Parsers. |
| **5** | LR Parsing Techniques: SLR, CLR, LALR, YACC/Bison. |
| **6** | Syntax-Directed Translation (SDT): SDD, Attributes, Dependency Graphs. |
| **7** | Intermediate Code Generation I: Three-Address Code, Quadruples, Triples. |
| **8** | Intermediate Code Generation II: Assignment, Boolean Expressions, Control Flow. |
| **9** | Run-Time Environments: Storage Organization, Activation Records, Heap Management. |
| **10** | Symbol Tables: Data Structures, Scope Management. |
| **11** | Code Optimization I: Basic Blocks, DAG, Local Optimization. |
| **12** | Code Optimization II: Loop Optimization, Data Flow Analysis, Peephole. |
| **13** | Code Generation: Target Machine, Register Allocation, Simple Code Generator. |

**Recommended Books:**
1. *Compilers: Principles, Techniques, and Tools* (The Dragon Book) by Aho et al.
2. *Compiler Design in C* by Allen I. Holub
