# ⚙️ CSTE-3107: Compiler Construction

Welcome to the Compiler Construction repository for the 5th semester.

## 🎯 Course Objectives
* Provide a deep dive into the phases of modern compiler design.
* Understand Lexical Analysis and the role of regular expressions and finite automata.
* Explore Syntax Analysis, covering Top-Down Parsing (LL(1)) and Bottom-Up Parsing (SLR, CLR, LALR).
* Learn Syntax-Directed Translation, Intermediate Code Generation (Three-Address Code), and Symbol Table management.
* Study code optimization techniques, run-time environments, and machine code generation.

## 📚 Recommended Reference Materials
1. **Compilers: Principles, Techniques, and Tools** (The Dragon Book) by Aho, Lam, Sethi, and Ullman.
2. **Compiler Design in C** by Allen I. Holub.

## 💻 Lab Setup & Projects
The laboratory component (`CSTE-3108`) involves practically building language translation tools.
* **Primary Language/Tools:** C/C++, Flex (Lex), and Bison (Yacc).
* **Key Tasks:** Designing basic scanners using Regular Expressions and DFAs, writing lexer/parser rules for simple languages, and producing intermediate code formats.
