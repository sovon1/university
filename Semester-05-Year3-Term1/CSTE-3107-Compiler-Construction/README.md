**Credits:** 3.0

## Course Overview
Detailed study of the phases of a compiler:
1.  **Lexical Analysis:** Scanning, Tokenizing, Regular Expressions.
2.  **Syntax Analysis:** Context-Free Grammars, Top-down (LL), Bottom-up (LR, LALR) parsing.
3.  **Semantic Analysis:** Type checking, Syntax-Directed Translation.
4.  **Intermediate Code Generation:** Three-address code.
5.  **Code Optimization:** Basic blocks, flow graphs.
6.  **Code Generation:** Register allocation.

## Lab (CSTE-3108)
Implementation of a subset of C compiler using Lex and Yacc tools.
