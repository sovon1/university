# ⚙️ CSTE 3107: Compiler Construction
**Credit:** 3.0 (Theory) + 1.0 (Lab) = 4.0 Total | **Prerequisite:** CSTE 2107

> *"Understanding compiler phases, automata theory, and constructing a compiler."*

</div>

---

## 🎯 Course Objectives
*   Provide a deep dive into the phases of modern compiler design.
*   Understand Lexical Analysis and the role of regular expressions and finite automata.
*   Explore Syntax Analysis, covering Top-Down Parsing (LL(1)) and Bottom-Up Parsing (SLR, CLR, LALR).
*   Learn Syntax-Directed Translation, Intermediate Code Generation, and Symbol Table management.
*   Study code optimization techniques, run-time environments, and machine code generation.

## 📅 Weekly Lesson Plan

| Week | Course Contents |
| :---: | :--- |
| **1** | Intro to Compilers: Phases (Lexical, Syntax, etc.), Tools. |
| **2** | Lexical Analysis: Tokens, Regular Expressions, Finite Automata, Lex/Flex. |
| **3** | Syntax Analysis (Parsing) I: CFG, Parse Trees, Top-Down Parsing (LL(1)). |
| **4** | Syntax Analysis (Parsing) II: Bottom-Up Parsing, Shift-Reduce, LR Parsers. |
| **5** | LR Parsing Techniques: SLR, CLR, LALR, YACC/Bison. |
| **6** | Syntax-Directed Translation (SDT): SDD, Attributes, Dependency Graphs. |
| **7** | Intermediate Code Generation I: Three-Address Code, Quadruples, Triples. |
| **8** | Intermediate Code Generation II: Assignment, Boolean Expressions, Control Flow. |
| **9** | Run-Time Environments: Storage Organization, Activation Records, Heap Management. |
| **10** | Symbol Tables: Data Structures, Scope Management. |
| **11** | Code Optimization I: Basic Blocks, DAG, Local Optimization. |
| **12** | Code Optimization II: Loop Optimization, Data Flow Analysis, Peephole. |
| **13** | Code Generation: Target Machine, Register Allocation, Simple Code Generator. |

## 📚 Recommended Books
1. 📖 *Compilers: Principles, Techniques, and Tools* (The Dragon Book) by Aho et al.
2. 📖 *Compiler Design in C* by Allen I. Holub
