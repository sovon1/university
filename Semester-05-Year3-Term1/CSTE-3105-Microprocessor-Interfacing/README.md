# 🎛️ CSTE 3105: Microprocessor, Microcontroller and Interfacing
**Credit:** 3.0 (Theory) + 1.0 (Lab) = 4.0 Total | **Prerequisite:** CSTE 2207

> *"Intro to microprocessor architecture (8086), assembly, and peripheral interfacing."*

</div>

---

## 🎯 Course Objectives
*   Explore the fundamentals of microprocessor architecture (focusing heavily on the Intel 8086).
*   Learn addressing modes, instructions, and write programs in Assembly Language.
*   Understand memory organization, interrupt structures, and DMA concepts.
*   Gain knowledge in peripheral interfacing (such as the 8255A PPI, 8254 Timer, and 8259A PIC).
*   Introduction to the 8051 Microcontroller architecture and modern embedded microcontrollers.

## 📅 Weekly Lesson Plan

| Week | Course Contents |
| :---: | :--- |
| **1** | Microprocessor Fundamentals: 8086 Architecture (BIU, EU), Registers. |
| **2** | Memory Organization & Addressing: Segmentation, Physical Memory, Pin Diagram. |
| **3** | Addressing Modes & Instruction Formats: Data, Program, Stack addressing. |
| **4** | Assembly Language Programming (Data Transfer): MOV, PUSH, POP, String Instructions. |
| **5** | Arithmetic & Logic Instructions: ADD, SUB, MUL, DIV, Logic, Shift/Rotate. |
| **6** | Program Control Instructions: Jumps, Loops, Procedures, Stack operations. |
| **7** | I/O Interfacing & Interrupts: I/O mapped vs Memory mapped, 8259A PIC. |
| **8** | Peripheral Interfacing I: 8255A PPI, Interfacing LEDs, Displays, Keyboards. |
| **9** | Peripheral Interfacing II: 8254 Timer, DMA Controller (8237). |
| **10** | Advanced Interfacing: ADC/DAC, Stepper Motor, RS-232C, USART. |
| **11** | Microcontrollers (8051): Architecture, Registers, I/O Ports, Memory. |
| **12** | 8051 Programming & Features: Instruction Set, Timers, Serial Comm, Interrupts. |
| **13** | Modern Microcontrollers: AVR (ATmega32), ARM, RISC features, Embedded Systems. |

## 📚 Recommended Books
1. 📖 *Microprocessors and Interfacing: Programming and Hardware* by Douglas V. Hall
2. 📖 *The Intel Microprocessors* by Barry B. Brey
