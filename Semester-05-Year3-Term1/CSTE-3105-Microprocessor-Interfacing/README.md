# 🎛️ CSTE-3105: Microprocessor, Microcontroller and Interfacing

Welcome to the Microprocessor and Interfacing repository for the 5th semester.

## 🎯 Course Objectives
* Explore the fundamentals of microprocessor architecture (focusing heavily on the Intel 8086).
* Learn addressing modes, instructions, and write programs in Assembly Language.
* Understand memory organization, interrupt structures, and DMA concepts.
* Gain knowledge in peripheral interfacing (such as the 8255A PPI, 8254 Timer, and 8259A PIC).
* Introduction to the 8051 Microcontroller architecture and modern embedded microcontrollers like AVR/ARM.

## 📚 Recommended Reference Materials
1. **Microprocessors and Interfacing: Programming and Hardware** by Douglas V. Hall.
2. **The Intel Microprocessors** by Barry B. Brey.

## 💻 Lab Setup & Projects
The laboratory component (`CSTE-3106`) is hardware and assembly-focused.
* **Primary Tools:** Emu8086 Emulator, Hardware Training Kits (e.g., MDE-8086), and optionally Proteus for simulations.
* **Key Tasks:** Writing 8086 Assembly code for data transfer, arithmetic, loops, and procedures. Building and programming basic interfaces for 7-segment displays, keyboards, and stepper motors.
