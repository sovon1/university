# CSTE-3106: Microprocessor Lab ⚙️

**Credits:** 1.5
**Kit:** MDE-8086 Trainer Board
**Tools:** TASM, EMU8086

## Lab Experiments
1.  **Familiarization:** Introduction to MDE-8086, Hex Keypad.
2.  **Basic I/O:** Reading keys, Displaying messages, Case conversion.
3.  **Arithmetic:** Addition, Subtraction, Multiplication (16-bit/32-bit).
4.  **Logic Operations:** AND, OR, XOR, Shift/Rotate.
5.  **Interfacing:**
    *   **8255 PPI:** LED Chasing, 7-Segment Display.
    *   **Stepper Motor:** Speed and Direction control.
    *   **Dot Matrix:** Displaying characters.
6.  **Advanced:** Interrupt handling, Macros, Sorting algorithms.
