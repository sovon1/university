# 🖥️ CSTE-3103: Operating Systems and System Programming

Welcome to the Operating Systems and System Programming course repository for the 5th semester.

## 🎯 Course Objectives
* Develop the ability to realize core Operating System functionalities.
* Cover process management, threads, and synchronization (semaphores, deadlocks).
* Understand memory management techniques like swapping, paging, and virtual memory.
* Explore file system implementations, mass-storage structures, and kernel I/O.
* Touch upon protection, security, and cryptography fundamentals.

## 📚 Recommended Reference Materials
1. **Operating System Concepts** by Silberschatz, Galvin, and Gagne.
2. **Modern Operating Systems** by Andrew S. Tanenbaum.

## 💻 Lab Setup & Projects
The laboratory component (`CSTE-3104`) revolves around kernel development, system calls, and scripting.
* **Primary Language/Environment:** C (System Calls), POSIX Threads (pthreads), and Linux Shell Scripting (Bash).
* **Key Tasks:** Implementing process scheduling algorithms, deadlock avoidance (Banker’s algorithm), page replacement algorithms, and multi-threading projects.
