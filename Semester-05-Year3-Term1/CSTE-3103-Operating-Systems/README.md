# 🖥️ CSTE 3103: Operating Systems and System Programming
**Credit:** 3.0 | **Prerequisite:** CSTE 2103

> *"Develops ability to realize OS functionalities (process, memory, file systems, security)."*

</div>

---

## 🎯 Course Objectives
*   Develop the ability to realize core Operating System functionalities.
*   Cover process management, threads, and synchronization.
*   Understand memory management techniques and virtual memory.
*   Explore file system implementations and mass-storage structures.
*   Touch upon protection, security, and cryptography fundamentals.

## 📅 Weekly Lesson Plan

| Week | Course Contents |
| :---: | :--- |
| **1** | Intro to OS: Types (Mainframe, Desktop, Real-time), OS Structures, System Calls. |
| **2** | Process Management: Process concept, Scheduling, IPC, Threads (Pthreads). |
| **3** | CPU Scheduling: FCFS, SJF, Priority, Round Robin, Multilevel Queue. |
| **4** | Process Synchronization: Critical-Section, Semaphores, Dining-Philosophers, Monitors. |
| **5** | Deadlocks: Characterization, Prevention, Avoidance (Banker’s Algo), Detection. |
| **6** | Memory Management: Swapping, Paging, Segmentation, Fragmentation. |
| **7** | Virtual Memory: Demand Paging, Page Replacement (FIFO, LRU), Thrashing. |
| **8** | File System Interface: Access Methods, Directory Structure, Protection. |
| **9** | File System Implementation: Allocation Methods, Free-Space Management, Recovery. |
| **10** | Mass-Storage Structure: Disk Scheduling, RAID Structure (Levels 0, 1, 5, 10). |
| **11** | I/O Systems: Polling, Interrupts, DMA, Kernel I/O Subsystem. |
| **12** | Protection and Security: Access Matrix, Threats (Viruses, Worms, DoS). |
| **13** | Cryptography & System Programming: Authentication, Assemblers, Linkers, Macro Processors. |

## 📚 Recommended Books
1. 📖 *Operating System Concepts* by Silberschatz, Galvin, Gagne
2. 📖 *Modern Operating Systems* by Andrew S. Tanenbaum
