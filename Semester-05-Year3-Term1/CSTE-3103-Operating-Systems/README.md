## Course Syllabus
*   **Introduction:** OS structures, kernel types, system calls.
*   **Process Management:** Processes, Threads, IPC, CPU Scheduling.
*   **Synchronization:** Critical section, Mutex, Semaphores.
*   **Deadlocks:** Prevention, Avoidance (Banker's Algorithm), Detection.
*   **Memory Management:** Paging, Segmentation, Virtual Memory.
*   **Storage Management:** File Systems, Disk Scheduling (RAID).
*   **Protection & Security:** Access Matrix, Authentication.
